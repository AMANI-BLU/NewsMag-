# NewsMag - React News Application 📰

NewsMag is a responsive React-based news application that fetches the latest headlines and displays categorized articles. It features a carousel for top news and a search functionality to quickly find specific topics. 

## Features 🌟
- **Top Headlines Carousel**: Highlights the latest news in a visually appealing carousel. 🎠
- **Category Filtering**: Browse articles by categories such as Technology, Health, Sports, Politics, Business, and Entertainment. 🗂️
- **Search Functionality**: Quickly search for articles using keywords. 🔍
- **Responsive Design**: Optimized for mobile and desktop users. 📱💻
- **Powered by News API**: Fetches the latest and trending articles in real time. 📰

## Technologies Used ⚙️
- **React**: Frontend framework. ⚛️
- **Axios**: For HTTP requests. 🌐
- **Bootstrap**: Styling and responsiveness. 🎨
- **React Icons**: For sleek iconography. 🖼️
